# VelaFlow Workflow

VelaFlow is designed around a release workflow:

1. Create or import song lyrics.
2. Use Director to plan storyboard scenes and emotions.
3. Generate or manually attach image assets.
4. Review and approve images.
5. Preview motion/subtitle styles.
6. Render draft, then final outputs.
7. Generate clips and Spotify Canvas-style loops.
8. Build marketing package.
9. Build final release package.
10. Review manually before upload.

No auto upload is performed.
