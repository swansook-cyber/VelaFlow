# VelaFlow Roadmap

## Current Focus
- Keep VelaFlow V1 as an AI Creative Pack Generator.
- Improve release pack quality for Suno/Udio and external AI video tools.
- Maintain simple creator flows for:
  - Song packages
  - Visual prompt packages
  - Character prompt packs
  - Affiliate creator packages
  - Optional mastered audio exports
- Stabilize closed beta UX and exports.

## Next Phase
- Polish Thai commercial title, hook, and lyric quality.
- Improve copy-ready prompt formatting for Flow, Veo, Kling, Runway, Pika, Whisk, and CapCut workflows.
- Improve package filenames, ZIP structures, and mobile copy/download UX.
- Add more high-quality creator presets without adding render complexity.
- Make project list, recent projects, and package history easier for beta users.

## Future Phase
- Add optional, reliable integrations only after V1 package generation is stable.
- Expand creator package templates for music, podcast, affiliate, MV, and character workflows.
- Improve local analytics for export usage and beta feedback.
- Add stronger quality reports for lyrics, hooks, prompts, and release readiness.

## Long-Term Vision
VelaFlow should become a creator operating system for AI-native music and short-form content: a place to shape ideas, direct creative outputs, organize reusable assets, and export clean packages into the best external production tools.
